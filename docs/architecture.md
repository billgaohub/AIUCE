# 十一层架构设计文档

## 概述

十一层架构是一种受中国古代官僚体系启发的AI系统设计模式。每一层都有明确的职责和边界，通过规范的消息传递机制进行协作。

## 核心思想

1. **分层治理**：每层独立负责特定领域
2. **制衡机制**：L0和L1拥有否决权
3. **数据流清晰**：输入→处理→输出路径明确
4. **可审计性**：所有决策都有日志记录
5. **可演化性**：L7负责系统自我改进

## 层级详解

### L0 意志层 (秦始皇/御书房)
- **职责**：最高宪法，一票否决权
- **输入**：所有上层输出
- **输出**：通过/否决
- **机制**：合宪性检查

### L1 身份层 (诸葛亮/宗人府)
- **职责**：人设边界，防止越权
- **输入**：用户请求
- **输出**：边界检查结果
- **机制**：人设一致性检查

### L2 感知层 (魏征/都察院)
- **职责**：现实对账，只说真话
- **输入**：外部数据
- **输出**：结构化感知数据
- **机制**：数据验证和意图识别

### L3 推理层 (张良/军机处)
- **职责**：多路径推演，后果分析
- **输入**：感知数据+记忆
- **输出**：推理结果和置信度
- **机制**：25个心智模型并行推演

### L4 记忆层 (司马迁/翰林院)
- **职责**：全域语义索引，史料编纂
- **输入**：需要存储/检索的内容
- **输出**：相关记忆
- **机制**：语义搜索和向量存储

### L5 决策层 (包拯/大理寺)
- **职责**：决策存证，审计落槌
- **输入**：推理结果
- **输出**：决策结果+审计ID
- **机制**：决策记录和风险评估

### L6 经验层 (曾国藩/吏部)
- **职责**：复盘机制，偏离度扫描
- **输入**：执行结果
- **输出**：经验模式
- **机制**：每日复盘和成功模式固化

### L7 演化层 (商鞅/中书省)
- **职责**：内核重构，物理变法
- **输入**：经验模式
- **输出**：系统更新
- **机制**：规则演化和权重调整

### L8 接口层 (张骞/礼部)
- **职责**：算力外交，模型调用
- **输入**：处理后的请求
- **输出**：模型响应
- **机制**：多模型路由和负载均衡

### L9 代理层 (韩信/锦衣卫)
- **职责**：跨设备执行，物理调度
- **输入**：决策指令
- **输出**：执行结果
- **机制**：工具调用和异步执行

### L10 沙盒层 (庄子/钦天监)
- **职责**：影子宇宙，模拟推演
- **输入**：决策方案
- **输出**：模拟结果
- **机制**：蒙特卡洛模拟和A/B测试

## 消息总线

层间通信通过异步消息总线进行：

```python
class MessageBus:
    def send(source, target, msg_type, payload)
    def subscribe(msg_type, handler)
    def broadcast(msg_type, payload)
```

消息类型定义在 `constants.MsgType` 中。

## 否决机制

L0 和 L1 拥有否决权：

```python
# L0: 违反宪法
if not constitution.is_constitutional(input):
    return VETO("违反宪法条款")

# L1: 人设越界
if identity.check_boundary(input).blocked:
    return VETO("人设边界限制")
```

否决后流程立即终止，返回否决原因。

## 审计日志

所有决策都记录到不可篡改的审计日志：

```python
audit.log_decision(
    input=user_input,
    decision=decision,
    reasoning=reasoning_result
)
```

日志包含：
- 时间戳
- 决策ID
- 输入内容
- 推理过程
- 决策结果
- 执行结果

## 扩展指南

### 添加新的心智模型 (L3)

```python
class MyMindModel:
    def analyze(self, input, context):
        # 自定义分析逻辑
        return {"confidence": 0.8, "result": ...}

# 注册到推理层
reasoning.register_model("my_model", MyMindModel())
```

### 添加新的数据源 (L2)

```python
class MyDataSource:
    def fetch(self):
        # 获取数据
        return data

perception.add_source("my_source", MyDataSource())
```

### 添加新的工具 (L9)

```python
class MyTool:
    def execute(self, params):
        # 执行操作

```

## 性能考虑

1. **异步处理**：L10沙盒模拟可异步执行
2. **缓存**：L4记忆层使用向量缓存
3. **批处理**：L3推理层可批量处理
4. **懒加载**：非必要层级按需初始化

## 安全考虑

1. **输入验证**：L2层负责清洗输入
2. **沙盒隔离**：L10提供隔离执行环境
3. **审计追踪**：所有操作可追溯
4. **权限控制**：L1层管理执行权限

## 未来方向

1. **分布式部署**：各层可独立扩展
2. **联邦学习**：L7演化支持分布式训练
3. **多模态**：扩展感知层支持图像/音频
4. **强化学习**：L6经验层结合RL优化
